## Tests

```
npm install
npm test
```

## Bundle

To prepare a browser-ready bundle, run the following:

```sh
npm run build

# Wrote dist/app.js
# Wrote dist/app.min.js
```

## Docs

Genarate human-friendly docs from the schema

```sh
npm run docs

# Wrote dist/schema.md
# Wrote dist/schema.html
```
